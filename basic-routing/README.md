# Basic Routing Configuration

## Objective
Configure routing between multiple IP networks to allow communication between
devices on different subnets.

## Environment
- Router and switch lab environment
- Static IP addressing

## Tasks Performed
- Assigned IP addresses to router interfaces
- Configured basic routing
- Verified connectivity using ping

## Skills Demonstrated
- IP addressing
- Routing fundamentals
- Network troubleshooting

## What I Learned
This lab improved my understanding of how routers forward traffic between
different networks and how to troubleshoot connectivity issues.
